Buttons

Buttons make common actions immediately visible and easy to perform with one click or tap.

Types

Sizes

Usage

Buttons are tools the user has to communicate something to the app and perform a desired action. In this system they are a Presentational React Components that you can use by simply instantiate by creating a DSButton tag and passing attributes for type (string), size (string), and a label (string). You can also use a Circle Button by instantiating a DSCircleButton tag and passing attributes for icon (string) and type (string). Check out the Iconography page as well.

<DSButton 
  label="save"
  type="primary"
  size="fluid"
  onClick="doSomething"
/>

<DSCircleButton 
  icon="face" 
  type="primary" 
/>
